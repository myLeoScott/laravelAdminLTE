# laravelAdminLTE

基于laravel5.5使用adminLTE 模板做的一个后台架子项目

# 使用的环境

+ laravel5+
+ php7+
+ bower
+ AdminLTE

# 使用方法

项目clone下来后。按顺序执行即可

1. 复制 `.env.example` 为 `.env`.在`.env`中配置自己的数据库

2. 执行 `composer update`

3. 执行 `php artisan key:generate`

4. 执行 `php artisan serve`
